Live AR Puzzle Game
This is a real-time Augmented Reality puzzle game built with Python, OpenCV, and MediaPipe that transforms your live webcam feed into an interactive 3x3 sliding puzzle controllable through hand gestures. To play, simply launch the script, hold up an open palm to freeze and slice the video feed, then hover your finger over pieces for two seconds to select and swap them until the image is restored. The project is designed for high-resolution performance, requiring only a standard webcam and the installation of OpenCV, MediaPipe, and NumPy to run.
